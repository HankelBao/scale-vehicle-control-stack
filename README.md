# Scale Vehicle Control Stack

This is the control stack for a scale vehicle being developed by [Wisconsin Autonomous](https://wisconsinautonomous.org/) in preparation for the Formula Student Driverless competition.

Stay up to date with our technical info by following our [blog](https://www.wisconsinautonomous.org/blog).
